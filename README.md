# 📊 Business Sales Dashboard

## 📌 Project Overview

This project analyzes sales data to generate business insights using Python.

## 🛠 Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn

## 📈 Key Insights

* Total Sales Performance
* Sales by Category
* Sales by Region
* Monthly Sales Trends
* Top Products Analysis

## 📂 Files Included

* `analysis.ipynb` → Data analysis & visualization
* `SampleSuperstore.csv` → Raw dataset
* `final_sales_data.csv` → Cleaned dataset

## 🚀 Outcome

Built a complete sales dashboard showing KPIs and trends for decision-making.
